# oinori

## どんなアプリ
* お祈り通知を体験できるアプリです。
* おめでと通知をもちろん体験できます。

## 開発環境
xcode 10.2.1
swift5
iPhoneXS

## 使用したライブラリ等
* Realm
  * データ保存用
* paper-switch(https://github.com/Ramotion/paper-switch)
  * on/offスイッチのUIに採用
